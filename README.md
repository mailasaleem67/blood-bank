# blood-bank-system
# 🩸 Blood Bank Management System

A **Blood Bank Management System** developed using **MySQL** to efficiently manage blood donors, patients, hospitals, blood inventory, donation records, and blood requests. The project demonstrates database design, relationships, constraints, triggers, views, events, and SQL queries for real-world blood bank operations.

---

## 📌 Features

* Hospital management
* Admin management
* Donor management
* Patient management
* Blood group management
* Blood inventory management
* Donation records
* Blood request management
* Blood stock availability checking
* Expired blood unit detection
* Eligible donor identification
* Blood request approval
* Automatic stock updates using triggers
* Automatic expiry status updates using events
* Activity logging
* Reports using SQL queries and views
